To use this code:

To run the website:  
git clone https://github.com/achieven/my-pages.git  
cd my-pages  
npm i  
nodemon  
open browser at http://localhost:5000  

To run tests:  
Unit tests:  
mocha test/unitTests/  

Browser tests:  
nodemon  
node test/browserTests/browserTests.js
